## 🚦 Featured Project — Traffic Sign Classification (GTSRB)

A comprehensive ML pipeline that benchmarks multiple model architectures on the German Traffic Sign Recognition Benchmark dataset.

| Architecture | Description |
|---|---|
| 🧱 **CNN** | Convolutional Neural Network for spatial feature extraction |
| 🔁 **RNN** | Recurrent Neural Network applied to sequential image features |
| 🤖 **Transformer** | Attention-based model for global context understanding |

> Goal: Compare accuracy, training time, and generalization across architectures on real-world traffic sign data.
